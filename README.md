
بوابة التوجيه - نسخة محسنة بالتصميم
----------------------------------
تغييرات رئيسية:
- لوحة ألوان مهنية (ذهبي، أزرق بحري، بيج).
- أزرار ذهبية مع تأثيرات ظل.
- شريط تنقل ثابت ومضبب عند التمرير.
- بطاقات تفاعلية مع تأثير رفع hover.
- قسم منصة فيديو متجاوب مع معاينات YouTube.
- تأثيرات دخول للعناصر (reveal on scroll) وطباعة احترافية.

تشغيل:
افتح index.html في متصفح. ملفات موجودة في assets/style.css و assets/script.js
